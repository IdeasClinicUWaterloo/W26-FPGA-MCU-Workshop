# Welcome to the W26 FPGA workshops!

This workshop is hosted by the [University of Waterloo IDEAS Clinic](https://uwaterloo.ca/engineering-ideas-clinic/) for January 2026.

You will learn what an FPGA is, how they are programmed, and why they are used.
You will create digital circuits, then see how Pong is realized on a FPGA.
